# MOLM1 Dashboard

Interactive Streamlit dashboard for analyzing chromatin interaction data in MOLM-1 leukemia cells.

## Live App
[Open the app](https://molm1-dashboard-gsrnehugqmu5nrvhgauumw.streamlit.app/)

## Features
- Upload CSV or Excel files
- Analyze chromatin interactions
- Interactive plots and filters
- Compare treatment-related patterns

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```
